### NEAR NRC20

## Installation:

python3.10+ is required.
1. `git clone https://github.com/cyberomanov/near-nrc20`
2. `cd ~/near-nrc20`
3. `pip3 install -r requirements.txt`
4. `nano config.py`
5. `python3 main.py`
